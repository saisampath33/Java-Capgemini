Problems:
Most methods are deprecated
Poor design
Not thread-safe

‚û§ Calendar

üëâ Because of these issues, Java 8 introduced a new API.

2Ô∏è‚É£ Modern Date & Time API (Java 8+) ‚Äî java.time

This is clean, immutable, and thread-safe.

Main classes:

üïí LocalDate ‚Äì Date only

Stores year, month, day.

System.out.println(today);


Example:

2026-01-26


Create specific date:

LocalDate dob = LocalDate.of(2002, 5, 10);

‚è∞ LocalTime ‚Äì Time only

Stores hour, minute, second.

üìÜ LocalDateTime ‚Äì Date + Time

Combination of both.

üåç ZonedDateTime ‚Äì Date + Time + Time Zone

Useful for global applications.

‚è≥ Duration vs Period

Used to calculate differences.

‚û§ Period (dates)
Period p = Period.between(d1, d2);


Measures in years, months, days.

‚û§ Duration (time)
Duration d = Duration.between(t1, t2);


Measures in hours, minutes, seconds.

üßæ Formatting Date & Time

Using DateTimeFormatter:

    DateTimeFormatter.ofPattern("dd/MM/yyyy");

Output:

üîÑ Parsing String to Date
