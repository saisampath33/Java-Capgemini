FileReader is a class used to read character data from a file.

Reads one character at a time
Used for text files
Slower if file is large (because char-by-char reading)

FileReader fr = new FileReader("sample.txt");
int ch;
while((ch = fr.read()) != -1){
    System.out.print((char)ch);
}
fr.close();

fr.read() returns:
an int representing ASCII/Unicode character
-1 means end of file (EOF)

BufferedReader reads text from a file efficiently by storing data in a buffer.

Faster than FileReader
Can read line by line
Best for big files

BufferedReader br = new BufferedReader(new FileReader("sample.txt"));
String line;
while((line = br.readLine()) != null){
    System.out.println(line);
}
br.close();

readLine() returns:
one full line
null means end of file
